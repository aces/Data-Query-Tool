Help: Define Filters

Filters can be applied on any field in order to limit the set of results returned by a query, e.g. to extract data collected only at one specific site, or from a particular visit.

The "Define Filters" tab is organized like "Define Fields", where data fields are grouped per instrument or category. Select a category, then click on a field of interest to add it to the list of Selected Filters at bottom.
Finally, in the Selected Filters panel, define the filter criteria on that field via the "Operator" and "Value" columns. More than one filter may be added to a query. Multiple filters will be applied using 'and' logic.
The operators are as follows:

  • = equal to
  • != does not equal
  • <= less than
  • >= greater than
  • startsWith: filter for values starting with a specific character or string
  • contains: filter for values containing a specific character or string
