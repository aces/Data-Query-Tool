Help: View Data

The View Data tab executes queries, displays query results, and enables download of data.

Under "Actions", click the "Run Query" button to execute the query on the database.


Results can be also sorted by field by clicking on the table's column headers. The number of results displayed per page can be modified, and keywords can be entered in the Search box.

After running a query and viewing the results, click the "Download Table Data as CSV" button under "Actions" to save the query output dataset to your computer as a comma-separated value file.
If any files are included in the query output, click "Download All Files as ZIP" to save compressed packages of the files and data to your computer.
The query can also be saved for reuse, to avoid constructing the query from scratch in future. Click the "Save Query" button, and a pop-up dialog will appear, asking whether the fields and/or filters should be saved. Click all checkboxes that apply, enter a name for the query, then click "Save query" to complete the process.

This saved query should now appear under the "Home" tab. New fields can be selected and saved under the same name to directly update the current query. See the section on "Saved Queries" for further information on actions that can be performed on these previously created queries.
